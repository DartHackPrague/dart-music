Drop your mp3 file here!
You will see some dart magic

Description

You can Drag and Drop your own mp3 file to play it instead of sample track.

By selecting a part of frequency chart, you can filter different frequencies in real time (only selected frequencies will be played). Hit spacebar for reseting frequency filter.

Canvas tag is used for animations and HTML5 audio API for reading live audio from played track.
Developed and tested in Chrome 18.0.1025.165.
